# Projeto ByteBank

## Requisitos funcionais

- Precisa lidar com transações bancárias
- Precisa lidar com contas bancárias
- Contas bancárias devem ter:
    - [ ] saque
    - [ ] depósito
    - [ ] transferência

## Requisitos não-funcionais

- Extrato bancário
- Impedir transferência para conta bloqueada
